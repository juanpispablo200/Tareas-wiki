Datos Abstractos

Tipos de Datos Abstractos

Existen varios tipos de datos abstractos comunes que se utilizan en la programación y el análisis de algoritmos. Algunos de los más conocidos son:

  • Listas: Una lista es una colección ordenada de elementos donde cada elemento tiene una posición única.
  • Pilas: Una pila es una colección de elementos donde el último elemento en entrar es el primero en salir (LIFO - Last In, First Out).
  • Colas: Una cola es una colección de elementos donde el primer elemento en entrar es el primero en salir (FIFO - First In, First Out).
  • Árboles: Un árbol es una estructura jerárquica que consta de nodos conectados entre sí por enlaces.
  • Grafos: Un grafo es una colección de nodos (vértices) conectados entre sí por aristas (arcos).

Ejemplo de Código

Aquí tienes un ejemplo de implementación de una lista en lenguaje Python:


class Lista:
    def __init__(self):
        self.items = []

    def esta_vacia(self):
        return len(self.items) == 0

    def agregar(self, item):
        self.items.append(item)

    def remover(self, item):
        if item in self.items:
            self.items.remove(item)

    def obtener_tamanio(self):
        return len(self.items)

    def imprimir(self):
        for item in self.items:
            print(item)

# Uso de la lista
mi_lista = Lista()
mi_lista.agregar(1)
mi_lista.agregar(2)
mi_lista.agregar(3)
mi_lista.imprimir()
